# Ejercicios JS  Roadmap

## Arrays (20 ejercicios)

### 1

Dado:

``` js
const numeros = [4, 10, 2, 8, 15, 1];
```

Filtra los números mayores que 5.

### 2

Usando el mismo array, crea un nuevo array donde cada número se
multiplique por 3.

### 3

Ordena el siguiente array de menor a mayor:

``` js
[9,3,7,1,6]
```

### 4

Ordena el mismo array de mayor a menor.

### 5

Encuentra el primer número mayor que 10.

``` js
[3,7,12,5,20]
```

### 6

Comprueba si existe algún número negativo.

``` js
[5,7,-2,9]
```

### 7

Comprueba si todos los números son positivos.

``` js
[3,8,11,2]
```

### 8

Convierte este array de strings a mayúsculas:

``` js
["hola","mundo","javascript"]
```

### 9

Suma todos los números del array:

``` js
[5,10,15]
```

### 10

Cuenta cuántos números son pares:

``` js
[1,2,3,4,5,6,7,8]
```

### 11

Elimina los duplicados:

``` js
[1,2,2,3,4,4,5]
```

### 12

Convierte este array en un string separado por comas:

``` js
["manzana","pera","uva"]
```

### 13

Devuelve solo los strings con más de 4 letras.

### 14

Ordena palabras por longitud.

### 15

Obtén los 3 números más grandes de:

``` js
[5,99,34,12,76,2,88]
```

### 16

Suma solo números pares.

### 17

Crea un array de cuadrados de:

``` js
[1,2,3,4,5]
```

### 18

Encuentra el índice del número 30:

``` js
[10,20,30,40]
```

### 19

Combina dos arrays:

``` js
[1,2,3]
[4,5,6]
```

### 20

Invierte el array:

``` js
[1,2,3,4,5]
```

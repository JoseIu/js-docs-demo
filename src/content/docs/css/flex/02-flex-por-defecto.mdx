import FlexDefault from '@components/css/flex/FlexDefault.astro';


El comportamiento por defecto de los elementos flexibles es el siguiente:

Como puedes ver, los elemetos hijos del contenedor `first-container` estan uno debajo del otro, pero que pasa si le damos un `display: flex` al contenedor padre, es decir a `first-container`.



Como puedes ver estos se ponen uno al lado del otro `main axis` y ocupan todo el alto del contenedor padre `cross axis`, esto es el comportamiento por defecto de los elementos. Para solucionar este comportamiendo `Si es algo que no queremos`, bastaría con poner un alto a los elementos hijos, es decir a `first-container__item` y ya no ocuparían todo el alto del contenedor padre.


```html
<div class="flex-default">
  <div class="first-container">
    <div class="first-container__item">1</div>
    <div class="first-container__item">1</div>
  </div>
</div>
```

```css
  .flex-default {
    padding: 1rem;
    background-color: #eee;
  }

  .first-container {
    width: 100%;
    height: 400px;

    border: 1px solid red;
    color: #222;
  }

  .first-container__item {
    width: 200px;
    padding: 1rem;
    list-style: 200px;
    text-align: center;
    border: 1px solid blue;
    background-color: #ccc;
  }
```

📝 **Resumen**

<img src="https://res.cloudinary.com/dz209s6jk/image/upload/f_auto,q_auto,w_700/Challenges/ftdvklbtn6y7ydjxjjuj.jpg" />
Tu desafío es construir esta página de inicio y lograr que se vea lo más parecida posible al diseño.

Puedes usar cualquier herramienta que te guste para completar el desafío. Así que si tienes algo que te gustaría practicar, siéntete libre de intentarlo.

El único JavaScript requerido es para el interruptor de tema de color.

**Tus usuarios deberían poder:**

- Cambiar el tema de color según su preferencia
- Ver el diseño óptimo para la interfaz dependiendo del tamaño de pantalla de su dispositivo
- Ver los estados de hover y focus para todos los elementos interactivos de la página

Descarga el código inicial y revisa el archivo README.md. Esto proporcionará más detalles sobre el proyecto. El archivo style-guide.md es donde encontrarás los colores, fuentes, etc.
